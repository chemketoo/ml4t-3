Both strategies run the in sample test case by default. To run the out
sample test case, comment out the call to testPolicy() (without arguments)
in each .py file and uncomment the testPolicy() call that supplies arguments.

For Theoretically Optimal Strategy:

Run:

python TheoreticallyOptimalStrategy.py

Stats print to the console. Images saved to the project directory.

For Manual Strategy:

Run:

python ManualStrategy.py

Stats print to the console. Images saved to the project directory.
